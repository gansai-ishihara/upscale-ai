# App Store Review Notes

## 審査メモ（App Store Connect「審査に関する備考」に貼り付け）

本アプリ「UpScale AI」は、CoreML（Real-ESRGANモデル）を使用し、すべてのAI動画超解像処理を完全にオンデバイスで実行します。動画データが外部サーバーに送信されることは一切ありません。

- ログイン・アカウント登録は不要です
- 無料ユーザーは1日3本まで動画を処理できます。Pro（¥480買い切り）で処理回数無制限・広告削除になります
- ATT（App Tracking Transparency）は広告パーソナライズのために表示されますが、許可しなくてもアプリの全機能は問題なく動作します
- 動画の長さにより、処理に1〜3分程度かかる場合があります（最大60秒の動画に対応）
- 対応フォーマット: MP4, MOV, AVI

## デモ手順（審査担当者向け）

1. アプリを起動します
2. ホーム画面の「動画を選択」ボタンをタップします
3. フォトライブラリへのアクセス許可を求められたら「許可」を選択します
4. 任意の動画（60秒以内）を選択します
5. アップスケール倍率（2x または 4x）を選択します
6. 「処理開始」をタップします
7. 処理中はプログレスバーが表示されます（1〜3分程度）
8. 処理完了後、ビフォー・アフターの比較画面で結果を確認できます
9. 「カメラロールに保存」をタップして保存できます

### Pro購入の確認

1. 無料状態で3本の動画を処理します
2. 4本目を選択すると制限メッセージが表示されます
3. 設定画面から「Proにアップグレード」で購入画面を表示できます
4. ¥480の買い切り購入後、処理回数の制限が解除され広告が非表示になります

### ATTダイアログの確認

- 初回起動時にATTの許可ダイアログが表示されます
- 「Appにトラッキングしないように要求」を選択しても、すべての機能が正常に動作します
