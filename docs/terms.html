利用規約

最終更新日: 2026年3月14日

本利用規約（以下「本規約」）は、UpScale AI（以下「本アプリ」）の利用条件を定めるものです。本アプリをご利用いただくことで、本規約に同意したものとみなされます。

1. サービス内容

本アプリは、AI超解像技術（Real-ESRGAN）を用いて動画の画質を向上させるサービスを提供します。すべての処理はユーザーの端末内で実行され、動画データがサーバーに送信されることはありません。

2. サブスクリプション

  • Proプラン（月額）: ¥480/月 - 自動更新サブスクリプション
  • Proプラン（年額）: ¥3,800/年 - 自動更新サブスクリプション

サブスクリプションに関する重要事項:

  • お支払いはApple IDアカウントに請求されます。
  • 現在の期間終了の24時間前までにキャンセルしない限り、サブスクリプションは自動的に更新されます。
  • 自動更新の解除は、購入後にiOSの設定アプリ → Apple ID → サブスクリプションから行えます。
  • 無料トライアル期間がある場合、未使用分はサブスクリプション購入時に失効します。

3. 無料プランの制限

  • 1日あたり3本までの動画処理
  • 出力動画にウォーターマーク（透かし）が追加されます
  • 広告が表示されます

以下の行為を禁止します:

本アプリは「現状のまま」提供されます。AI超解像処理の結果は入力動画の品質やデバイスの性能によって異なり、特定の結果を保証するものではありません。
